## Dutch Housing Price Analysis

This project analyses long-term trends in Dutch housing prices using CBS data,
with a focus on national, regional, provincial, and municipal differences.

### Business Questions
- How have Dutch house prices evolved over the last 30 years?
- How do trends differ across regions and municipalities?
- Which areas experienced the fastest growth versus highest price levels?

### Approach
- Data cleaning and feature engineering
- Time-series trend analysis
- Regional and municipality-level comparisons
- Growth, volatility, and CAGR analysis

### Key Insights
- House prices show long-term upward trends with cyclical phases
- Strong regional disparities persist
- Fastest-growing areas are not always the most expensive

### Tools
- Python (pandas, matplotlib, seaborn)
